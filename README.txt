A.P. Painting Services LLC - Local Website

How to run locally:
1. Extract the ZIP.
2. Open the folder.
3. Double-click index.html.

Recommended structure:
- index.html
- styles.css
- script.js
- assets/
  - logo.png
  - hero.jpg
  - work1.jpg
  - work2.jpg
  - work3.jpg
  - work4.jpg

You can replace images inside assets/ with your own real project photos later.
Keep the same filenames or update the references in index.html.
